# 積読コーチ（執行アプリ）

企画書に基づく「管理ではなく執行」「OSレベルの導線短縮」を最優先したMVP。

## 起動

```bash
cd app
npm start
```

- Android: `npm run android` または Expo Go で QR 読み取り
- iOS: Expo Go で QR 読み取り（Mac の場合は `npm run ios`）

## 構成

- **初期画面 = 執行画面**: 今日の1冊・予約時刻・「今すぐ読む」CTA・救済（5分/1ページ）
- **明日の予約**: 1冊選択 + 時刻プリセット → 通知を予約時刻に発火
- **本を追加**: タイトル必須・著者任意のみ

データは AsyncStorage。通知タップでアプリ起動 → 執行画面がそのまま表示される（導線短縮）。
