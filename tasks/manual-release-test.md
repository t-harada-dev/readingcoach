## 実行手順

```bash
# シミュレータでのビルド＆起動
cd app/ios
xcodebuild -workspace app.xcworkspace -scheme app -configuration Debug \
  -destination 'platform=iOS Simulator,name=iPhone 17 Pro' build
xcrun simctl boot "iPhone 17 Pro" 2>/dev/null; open -a Simulator
xcrun simctl install booted ~/Library/Developer/Xcode/DerivedData/app-*/Build/Products/Debug-iphonesimulator/app.app
xcrun simctl launch booted com.toyotomisoncho.readingcoach

# Metro bundler（別ターミナル）
cd app && npx expo start
```

## 判定基準

- **全 44 項目 PASS**: リリース可
- **M-37〜M-44（エッジケース・UI）に1件でも FAIL**: 重要度を判断し、修正 or Known Issue として記録
- **M-01〜M-36（コア機能）に1件でも FAIL**: リリース不可、修正必須
